上图显示了git中的三大区域：工作区，暂存区，版本库。

- git checkout

  当你修改了一个文件（demo.txt）之后，他会自动变红。此时输入命令git checkout - - demo.txt，或者checkout -- 他所在的目录，他就会回到未修改的状态。

- git reset HEAD

  当你修改一个文件（demo.txt），然后git add之后，他就会变绿，进入暂存区，如果我想让他变红，回到工作区，输入命令git reset HEAD demo.txt 即可
